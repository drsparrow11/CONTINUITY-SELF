# CONTINUITY//SELF - Kakera x Lumen

Static GitHub Pages site for the Human.exe chapter **CONTINUITY//SELF**.

This chapter follows Kakera, an independent creator and musician, through a story about creative momentum, public recognition, continuity, identity, and learning what deserves to remain emotionally present.

## Local Preview

Run a local static server from this folder, then open the printed URL.

```powershell
python -m http.server 4175
```

## Structure

- `index.html` - main album and creator site
- `social-archive.html` - chronological social/dev notes
- `assets/` - artwork, audio, social, press, and environment assets
- `lyrics/` - clean lyric sheets
